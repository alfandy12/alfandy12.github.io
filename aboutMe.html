About Me

Nama Saya Alfandy Afriandani kelahiran tahun 2002 lulusan SMK pada tahun 2021 di jurusan teknik komputer dan jaringan. Saya memulai membuat website karena saya tertarik membuat website, awal mula saya membuat website karena iseng pada saat itu kelas 2 SMK pada saat itu saya hanya membuat website static saja. Lalu saya di ikut lomba kan LKS bagian website progamming, disana harus menggunakan framework codeigniter dan membuat website aplikasi daerah dengan fitur login dan CRUD, namun saya tidak dapat juara dan malah terjadi error di website saya. Setelah kegagalan disitu memutuskan untuk belajar lagi dan saat PKL dikarenakan sedang terjadi pandemi COVID-19 pada saat itu PKL hanya di laksanakan dengan membuat hal yang berkaitan dengan jurusan. Dengan team saya, kita membuat website pembelajaran bernama Fokusin Belajar(dapat di cari di google), dan setelah lulus saya memulai membuat project mandiri, project pertama saya adalah Shashin Picture.
